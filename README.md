# School Clubs Telegram Bot

Бот для управления школьными кружками и секциями в Telegram.

## Функциональность

- 📋 Просмотр списка доступных кружков
- ✅ Запись на кружки с учетом квоты мест
- 📝 Просмотр своих записей
- ❌ Отмена записи
- 👑 Административная панель
- 📊 Статистика посещаемости
- 📤 Выгрузка отчетов в CSV
